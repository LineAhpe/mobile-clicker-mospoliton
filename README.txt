1. Склонировать репозиторий:
git clone https://github.com/ВАШ_ЛОГИН/lab2-clicker.git
cd lab2-clicker


2. Установить зависимости:
npm install


3. Запустить приложение в Expo:
npm start


4. В открывшемся меню Expo:
нажать w для запуска web-версии или отсканировать QR-код через приложение Expo Go на телефоне.
